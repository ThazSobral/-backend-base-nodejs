# Backend-Base
### Modelo base para backend NodeJS

Para utilizar aplicar o comando `git clone https://github.com/ThazSobral/Rocketseat-Nodejs.git`.

---
#### Contém:

* routes
* controllers
* models
* cors
* json default
* CRUD
* paginate
* websocket

#### Características:
* node v12.0
* npm v6.9
* yarn v1.21.1
